# Lab.dev - Profile Link Platform

## Overview
Lab.dev is a customizable profile link platform.

## Features
- Personalized landing pages
- Social links & custom themes
- Interactive effects (GSAP/Framer Motion)
- Real-time profile preview
- Gamification (XP/Level/Views)

## Technical Stack
- **Frontend**: React, Vite, Wouter, TanStack Query, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express.js, Passport.js, PostgreSQL (Drizzle ORM)

## Latest Changes
- Implemented full authentication system (Passport.js local strategy)
- Added protected routes for /lab and /analytics
- Updated schema to support JSONB theme configuration
- Enhanced Lab UI with Save Changes and Logout features
- Optimized deployment configuration for autoscale
